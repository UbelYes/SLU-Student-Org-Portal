## Revision Features

### Form Return System
- **OSA Portal**: OSA staff can return submitted forms to organizations with a reason
- **Return Reason**: OSA provides specific feedback on why a form needs resubmission
- **Visual Indicators**: Returned forms are highlighted in yellow for easy identification
- **Status Tracking**: Forms display their status (Submitted/Returned) in both portals

### Notifications
- **Popup Alerts**: Organizations receive instant notifications when forms are returned
- **Form Details**: Notifications show which specific form(s) were returned

### PDF View

- **File Information**: Attached file names are shown in the document preview

### Table Header Sorting
- **Table Sorting**: Click column headers to sort data in both OSA and Organization portals


### Security Enhancements
- **.htaccess Protection**: Secure directory access and prevent unauthorized file browsing
